TaskFlow - Full Stack Task Manager

A full stack task management app built from scratch, no frameworks. Features JWT authentification with bcrypt password hashing, a FastAPI REST API with PostgreSQL, and dynamic vanilla JS frontend. Supports full task CRUD, priority and status filtering, overdue detection, a live dashboard summary, and an auth guard that protects all routes. Backend depoyed on Railway, frontend on GitHub Pages.

Asteroid Blaster

A fully featured arcade space shooter built in Python with Pygame. Pilot a ship through endless ateroid waves, asteroids split into smaller fragments on impact, difficulty scales over time, and the scoring system rewards larger kills. Features sprite-based rendering, circle collision detection, physics-based thrust and friction, sound effects, background music, and a web build via Pybag so it runs directly in the browser.

04

Fruit Collector

A fast-paced arcade browser game built in Python with Pygame. Control a basket to catch 7 types of falling fruit while dodging bombs that drain your score. Features timed independent spawn events, variable fall speeds, off-screen object culling, and a full win/lose/restart game loop. Compiled to WebAssembly with Pybag for instant browser play.

  • Python
  • Pygame
  • Pybag
